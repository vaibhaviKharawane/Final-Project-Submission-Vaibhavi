//Personal Finance Manager with MERN Technology 

HOW TO RUN Frontend
STEP 1: cd frontend
STEP 2: npm i
STEP 3:npm run dev
it is open in http://localhost:5173/

How to run Backend
Step 1:cd backend
Step 2:npm i
Step 3:npm run start
[nodemon] starting `node index.js`
Server listining on http://127.0.0.1:3001
